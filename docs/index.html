INTELLIGENT AUTOMATION TECH RADAR

O Intelligent Automation Tech Radar é uma lista de tecnologias, complementada por um resultado da avaliação. Estruturado por 4 anéis:

  • ADOTAR — Tecnologias de alto desempenho funcional, aplicável para o longo prazo e larga escala. São capazes de combinar uma compreensão perspicaz do mercado e lançar tendências.
  • TESTAR — Tecnologias que estão sendo implementadas com sucesso no mercado, com visão de longo prazo mas que ainda não conseguem entregar um alto desempenho funcional para larga escala. São capazes de desafiar as tendências de mercado, proporcionando disrupção.
  • AVALIAR — Tecnologias promissoras e com um potencial de agregação de valor, visão de longo prazo mas com dificuldades de entregar soluções escaláveis. São ótimos para iniciar esforços com prototipação.
  • HOLD — Tecnologias não recomendadas para serem usadas em novos projetos. As tecnologias HOLD não devem ser usadas para novos projetos, mas geralmente podem ser continuadas para projetos existentes.

PROPÓSITO

O Tech Radar é uma ferramenta para inspirar e apoiar as equipes de engenharia em Zalando na escolha das melhores tecnologias para novos projetos; fornece uma plataforma para compartilhar conhecimento e experiência em tecnologias, refletir sobre decisões tecnológicas e evoluir continuamente nosso cenário tecnológico. Com base no trabalho da ThoughtWorks.
